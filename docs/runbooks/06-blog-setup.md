# Runbook 06 — Blog Setup (Wix Blog)

**Audience:** owner / content. **Phase:** 3. **Goal:** an SEO-friendly, on-brand blog that supports
the events-first story.

## 1. Add Wix Blog
1. Add the **Wix Blog** app from the App Market.
2. Set the blog landing page at **/blog** and add it to the footer (not the crowded top nav).

## 2. Categories & structure
Create a small set of categories (`TODO(owner)` refine):
- **Event Recaps** (weddings, corporate, private) — doubles as social proof.
- **Drinks & Menus** — signature drinks, seasonal, alt-milk features.
- **Planning Tips** — "how much coffee for X guests," timelines, power/space needs.
- **Behind the Bar** — the team, sourcing, the craft.

## 3. Templates & cleanliness
1. Set a consistent **post template**: one hero image, clear headings, generous spacing.
2. Each post ends with a **CTA** to Book or Request a Quote (no dead ends).
3. Use the brand fonts/colors only; avoid mixed styles.

## 4. SEO basics
1. Set per-post SEO title + meta description + slug.
2. Add alt text to images.
3. Internal-link posts to Services / Menus / Book / Quote.

## 5. Cadence
- Define a realistic publishing cadence (`TODO(owner)`, e.g. 1–2/month). Maintain a content
  calendar in **Notion**, seeded from [`content/blog/content-calendar.md`](../../content/blog/content-calendar.md).

## ✅ Verify it worked
- /blog lists posts, reads cleanly on mobile, and isn't cluttering the top nav.
- Each post has SEO fields + a closing CTA.
- A content calendar exists in Notion.
